\begin{frame}{Проблематика}
%\pause
\setbeamercolor{block body}{bg=black, fg=white}
\begin{block}{Скорость построения изображения}
\vspace{1ex}
\hspace{1em}\parbox{21em}{
С каждой итерацией число точек изображения растет экспоненциально. Поэтому с увеличением степени детализации время построение изображения значительно увеличивается.
}\hspace{.0em}
\raisebox{-3ex}{
\includegraphics[width=5em, keepaspectratio]{beamer9_bib.PNG}
}
\vspace{1ex}
%}

\end{block}
\pause
\begin{block}{Решение}
\vspace{1ex}
\hspace{1em}\parbox{21em}{
Использование многопроцессорных или много-ядерных систем для вычисления точек изображения. Кроме того использовать фильтрацию точек в процессе вычисления.
}\hspace{.2em}
\raisebox{-3ex}{
\includegraphics[width=5em, keepaspectratio]{beamer8_rb.PNG}
}
\vspace{0ex}
%}

\end{block}

\end{frame}

\begin{frame}{Проблематика}
%\pause
\setbeamercolor{block body}{bg=black, fg=white}
\begin{block}{Хаотичность}
\vspace{1ex}
\hspace{1em}\parbox{21em}{
Небольшие изменения порождающих параметров могут привести к значительным изменениям изображения. Как следствие, сложно построить изображение с наперед заданными свойствами.
}\hspace{.0em}
\raisebox{-5ex}{
\includegraphics[width=5em, keepaspectratio]{beamer6_bib.PNG}
}
\vspace{1ex}
%}

\end{block}
\pause
\begin{block}{Решение}
\vspace{1ex}
\hspace{1em}\parbox{21em}{
Создание библиотеки шаблонов изображений с заданными характеристиками. Изучение влияния параметров, используемых при построении, на форму изображения.
}\hspace{.2em}
\raisebox{-5ex}{
\includegraphics[width=5em, keepaspectratio]{beamer5_rb.PNG}
}
\vspace{0ex}
%}

\end{block}
   
\end{frame}
