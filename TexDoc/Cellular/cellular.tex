\section*{Клеточные автоматы}
\begin{itemize}
\item Обработка изображений. Многократные итерации соответствуют свертке с большим ядром. Степенные фильтры

Множество фильтров, основанных на свертках имеет непустое пересечение с клеточными автоматами. Это так называемые степенные фильтры.

\item Обработка звука на видео-карте

Одномерная свертка соответствует произведению многочленов. Согласно основной теореме алгебры, над полем комплексных чисел многочлен, представляющий ядро свертки можно разложить в произведение линейных.
\item Преобработка изображения, выделение инвариантов.

Пример: разбиение плоскости на области.
\item Параллельные вычисления

Возможность использования формализма клеточных автоматов для реализации модели параллельных вычислений
\end{itemize}
