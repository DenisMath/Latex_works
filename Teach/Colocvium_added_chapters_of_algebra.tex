{\sl Додаткові розділи алгебри. Теорія груп.}

\begin{enumerate}
\item Група, аксіоматика, приклади груп.
\item Сенс асоціативності, приклади неасоціативних операцій
\item Доведення єдиності нейтрального, оберненого елемента в групі.
\item Обговорення некомутативності. Приклади.
\item Підгрупа. Означення, приклади.
\item Класи суміжності. Теорема Лагранжа.
\item Порядок елемента, породжуючий елемент.
\item Циклічна група, породжуючі елементи $\mathbb{Z}_n^+$.
\item Ізоморфізм груп. Приклади ізоморфних груп. Інваріанти груп при ізоморфізмних відображеннях.
\item Група підстановок, кількість елементів. Приклади роботи, множення елементів. Теорема Келі.
\item Нормальні підгрупи. Фактор-група, означення, приклади, кількість елементів.
\item Гомоморфізм груп. Приклади. Теорема о гомоморфізмі.
\item Прямий добуток, напівпрямий добуток, ланцюговий добуток. Означення, приклади. Доведення, що наведенні конструкції визначають групу.
\item Приклади відображень, що зберігають структуру.
\item Автоморфізми груп. Група автоморфізмів. Приклади.
\item Голоморф. Приклади.
\end{enumerate}
